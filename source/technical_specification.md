# Technical Specification - hackyourway.cz

## Technical Stack
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) (utility-first framework).
- **Fonts:** **Sora** (Geometric Sans-Serif).
- **Icons:** **Lucide Icons** (or similar SVG icons).

## Color Palette
| Name | Hex Code | Usage |
| :--- | :--- | :--- |
| **Primary Accent** | `#BAF122` | Buttons, highlights, hover states (Lime Green). |
| **Background (Deep)** | `#000000` | Primary background color. |
| **Background (Dark)** | `#0A0A0A` | Section backgrounds, cards (very dark gray). |
| **Text (Primary)** | `#FFFFFF` | Headings, main content. |
| **Text (Secondary)** | `#D1D5DB` | Descriptions, secondary info (`gray-300`). |

## Typography
- **Headings:** `Sora`, font-weight `800` (Extra Bold), `text-7xl` on large screens.
- **Body:** `Sora`, font-weight `400` (Regular), `text-lg` or `text-base`.
- **Labels:** `Sora`, font-weight `600` (Semi Bold), uppercase for small caps/tags.

## Asset Directory (Identified)
- **Logo:** Multi-color geometric font "HYW".
- **Hero Image:** Urban night shot with neon tram lights (long exposure).
- **Section Dividers:** Glowing gradients and subtle borders.
