# Layout Guide - hackyourway.cz

The layout is structured as a vertical stack of high-impact sections, optimized for scroll-based storytelling.

## 1. Header (Sticky)
- **Composition:** Logo (Left), Navigation Links (Center), CTA Button (Right).
- **Styling:** `sticky top-0`, `backdrop-blur-md`, `bg-black/50`.

## 2. Hero Section
- **Composition:** Large H1 heading -> Sub-headline -> CTA Group.
- **Visuals:** Full-screen width, dynamic background image with a dark overlay.

## 3. Info Grids (Mindset & Stats)
- **Mindset:** `grid-cols-1 md:grid-cols-3`. Large icons paired with short, punchy paragraphs.
- **Stats:** Simple horizontal flex/grid layout showing numbers and labels.

## 4. Program / Schedule
- **Structure:** Tabbed interface or vertical timeline.
- **Styling:** Dark background with subtle card outlines (`border-white/10`).

## 5. Speaker Cards (Disrupce)
- **Grid:** `grid-cols-2 lg:grid-cols-4` (or similar).
- **Interaction:** Zoom effect on hover, text overlays, high-contrast portraits.

## 6. Footer
- **Composition:** Secondary navigation, Social links, Legal info.
- **Styling:** Muted gray text, minimalist design.
